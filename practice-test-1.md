### 第四部分：英译汉（共 15 分）

1.  **A firewall is a network security system that monitors and controls incoming and outgoing network traffic based on predetermined security rules.**
    -   翻译：防火墙是一种网络安全系统，它根据预定的安全规则监视和控制传入和传出的网络流量。
2.  **The fundamental difference between machine language and high-level language is that machine language instructions are directly executable by the CPU.**
    -   翻译：机器语言和高级语言的根本区别在于机器语言指令可以直接被中央处理器执行。
3.  **A computer virus is a type of malicious software program that, when executed, replicates itself by modifying other computer programs and inserting its own code.**
    -   翻译：计算机病毒是一种恶意软件程序，它在执行时通过修改其他计算机程序并插入自己的代码来复制自身。
