### 第四部分：英译汉（共 15 分）

1.  **The World Wide Web (WWW) is a system of interlinked hypertext documents accessed via the Internet.**
    -   翻译：万维网（WWW）是一个通过互联网访问的、相互连接的超文本文件系统。
2.  **In computer science, a data structure is a particular way of organizing data in a computer so that it can be used efficiently.**
    -   翻译：在计算机科学中，数据结构是一种在计算机中组织数据的特定方式，以便数据可以被高效地使用。
3.  **Source code is any collection of computer instructions that can be compiled or assembled into an executable computer program.**
    -   翻译：源代码是任何一组计算机指令的集合，这些指令可以被编译或汇编成可执行的计算机程序。
