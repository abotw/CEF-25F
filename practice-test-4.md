### 第四部分：英译汉（共 15 分）

1.  **A compiler is a software program that transforms source code written in a programming language into another computer language, usually machine code.**
    -   翻译：编译器是一种将以编程语言编写的源代码转换成另一种计算机语言（通常是机器代码）的软件程序。
2.  **The topology of a network defines the physical or logical arrangement of the connected devices, which affects performance and cost.**
    -   翻译：网络拓扑结构定义了连接设备的物理或逻辑排列方式，这会影响性能和成本。
3.  **Debugging is the process of finding and resolving bugs or defects that prevent correct operation of computer software or a system.**
    -   翻译：调试是查找和解决阻止计算机软件或系统正确运行的程序错误或缺陷的过程。
