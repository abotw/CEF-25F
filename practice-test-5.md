### 第四部分：英译汉（共 15 分）

1.  **An operating system is the core software that manages computer hardware and software resources and provides common services for computer programs.**
    -   翻译：操作系统是管理计算机硬件和软件资源并为计算机程序提供通用服务的核心软件。
2.  **Data integrity refers to maintaining and assuring the accuracy and consistency of data over its entire life-cycle.**
    -   翻译：数据完整性指的是在其整个生命周期中维护和确保数据的准确性和一致性。
3.  **The main purpose of a compiler is to convert the program code into an executable file that can run directly on the hardware.**
    -   翻译：编译器的主要目的是将程序代码转换成可以直接在硬件上运行的可执行文件。
