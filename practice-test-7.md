## **第四部分：英译汉（共 15 分）**

1.  **Cloud computing enables users to access computing resources on demand without the need to manage physical hardware.**
    -   云计算使用户能够按需访问计算资源，而无需管理物理硬件。
2.  **A database transaction must ensure that all operations are completed successfully; otherwise, the entire transaction should be rolled back.**
    -   数据库事务必须确保所有操作都成功完成；否则整个事务应被回滚。
3.  **Efficient memory management helps the operating system allocate resources properly and avoid performance bottlenecks.**
    -   高效的内存管理有助于操作系统合理分配资源并避免性能瓶颈。
